# Pocket
## Liste Defteri Uygulaması

Basit ve anlaşılır yapıda hazırlanmış bir liste oluşturma ve yönetme uygulamasıdır. Kullanıcıların günlük notlarını, yapılacaklarını veya tutulması gereken bilgileri hızlı şekilde ekleyip düzenlemesini sağlar.

### Özellikler
- Yeni Maddeyi Düzenleyebilir, Silebilirsiniz.
- İstediğiniz Kadar Eleman Ekleyebilirsiniz.
- Temiz Bir Arayüze sahip.
